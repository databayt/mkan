# Story 5.5: Real hosting notifications

> **Epic**: 5 · **Status**: To Do · **Phase**: C

## Story
As a host, I want my notification card to surface real action items (pending bookings, upcoming check-ins, low-rated reviews), so that I know what to do today.

## Acceptance Criteria
1. `notification-card.tsx` reads from `getHostNotifications(userId)` action.
2. Action returns: pending bookings (≤5), upcoming check-ins (next 7 days), recent reviews < 4 stars.
3. Each item has a clickthrough link.
4. Empty state shows friendly "All caught up" message.
