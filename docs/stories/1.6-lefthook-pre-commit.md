# Story 1.6: Lefthook pre-commit hook

> **Epic**: 1 · **Status**: To Do · **Phase**: B

## Story
As a developer, I want pre-commit hooks (tsc + lint + format) to run on staged files, so that broken code never reaches `main`.

## Acceptance Criteria
1. `lefthook.yml` at repo root with pre-commit + pre-push stages.
2. Pre-commit runs: `prettier --check` on staged, `eslint` on staged, `tsc --noEmit`.
3. `lefthook install` runs in `postinstall` (or via README instruction).
4. A failing format/lint blocks the commit.
