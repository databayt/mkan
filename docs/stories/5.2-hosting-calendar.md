# Story 5.2: Hosting calendar

> **Epic**: 5 · **Status**: To Do · **Phase**: C

## Story
As a host, I want a month-grid calendar showing bookings + blocked dates per listing, so that I can manage my availability visually.

## Acceptance Criteria
1. `/[lang]/hosting/calendar/page.tsx` renders a month grid (use `react-day-picker` already a dep).
2. Listing selector at the top (defaults to most recent).
3. Cells show: empty / booked (color) / blocked (color).
4. Click a date → toggle block (calls `setBlockedDate`).
5. Replaces current stub.
