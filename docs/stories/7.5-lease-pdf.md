# Story 7.5: Lease PDF generation

> **Epic**: 7 · **Status**: To Do · **Phase**: C

## Story
As a tenant or manager, I want to download a PDF of the lease, so that we have a written record.

## Acceptance Criteria
1. New helper `src/lib/lease-pdf.tsx` using `@react-pdf/renderer`.
2. Route `GET /api/leases/[id]/pdf` returns `application/pdf`.
3. Auth: only tenant or manager on the lease can download.
4. Bilingual: lang param controls en/ar.
